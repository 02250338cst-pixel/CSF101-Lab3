# CSF Lab Assignment

## Aim of the Lab
The aim of this lab is to practice Python programming using conditional statements, loops, break/continue, and functions. It helps develop problem-solving skills and understanding of basic programming concepts.

## List of Programs
### Part A: Conditional Statements
1. Positive, Negative, or Zero  
2. Even or Odd  
3. Largest of Two Numbers  
4. Largest of Three Numbers  
5. Grade Calculation  
6. Menu-driven Calculator  

### Part B: Loops (for and while)
7. Print numbers 1 to 10 using a for loop  
8. Print even numbers from 1 to 20  
9. Sum of first 10 natural numbers  
10. Multiplication table of a given number  
11. Print numbers 1 to 5 using a while loop  
12. Sum from 1 to n using a while loop  
13. Keep taking input until user enters 0  

### Part C: Break and Continue
14. Print numbers 1 to 10 but stop at 6 (break)  
15. Print numbers 1 to 10 but skip 5 (continue)  
16. Search for a number in a list and stop when found  

### Part D: Functions
17. Function that prints “Hello, World!”  
18. Function to return the square of a number  
19. Function to add two numbers  
20. Function to check even or odd  
21. Function to calculate factorial  
22. Function to print multiplication table  

### Part E: Integrated Programs
23. Print numbers from 1 to n and display even/odd  
24. Menu-driven calculator using functions and loop

## Brief Explanation of Each Task

### Part A: Conditional Statements
- **Positive, Negative, or Zero:** Determines if a number is positive, negative, or zero using `if-elif-else`.  
- **Even or Odd:** Checks if a number is divisible by 2 to determine even or odd.  
- **Largest of Two Numbers:** Compares two numbers and prints the larger.  
- **Largest of Three Numbers:** Uses nested `if-else` to find the largest among three numbers.  
- **Grade Calculation:** Assigns grade A/B/C/Fail based on input marks.  
- **Menu-driven Calculator:** Performs addition, subtraction, multiplication, or division based on user choice.

### Part B: Loops
- **For loop 1–10:** Prints numbers from 1 to 10 using a `for` loop.  
- **Even numbers 1–20:** Prints all even numbers in the given range using step in `for`.  
- **Sum of first 10 numbers:** Calculates sum using a loop.  
- **Multiplication table:** Prints multiplication table of a given number.  
- **While loop 1–5:** Prints numbers using a `while` loop.  
- **Sum 1–n:** Uses `while` loop to calculate sum up to n.  
- **Input until 0:** Repeatedly takes input until user enters 0.

### Part C: Break and Continue
- **Stop at 6:** Uses `break` to exit the loop when a condition is met.  
- **Skip 5:** Uses `continue` to skip a specific number in loop iteration.  
- **Search in list:** Loops through list and stops when the number is found.

### Part D: Functions
- **Hello, World!:** Simple function to print a message.  
- **Square function:** Returns the square of a number.  
- **Addition function:** Adds two numbers and returns the sum.  
- **Even/Odd function:** Checks if input number is even or odd.  
- **Factorial function:** Calculates factorial using loop.  
- **Multiplication table function:** Prints multiplication table of a number.

### Part E: Integrated Programs
- **Even/Odd 1–n:** Prints numbers 1 to n and specifies if each is even or odd.  
- **Menu-driven calculator:** Uses functions and a loop to continuously perform calculations until user exits.
