# CSF101-Lab3

## Aim of the Lab
The aim of this lab is to practice writing Python programs using conditional statements and basic arithmetic operations.

## List of Programs
1. Positive, Negative, or Zero
2. Even or Odd
3. Largest of Two Numbers
4. Largest of Three Numbers
5. Grade Calculation
6. Menu-driven Calculator

## Brief Explanation of Each Task
1. **Positive, Negative, or Zero:** Checks if a given number is positive, negative, or zero using if-elif-else.
2. **Even or Odd:** Determines if a number is even or odd using the modulus operator.
3. **Largest of Two Numbers:** Compares two numbers and prints the larger one.
4. **Largest of Three Numbers:** Uses nested if-else to find the largest among three numbers.
5. **Grade Calculation:** Assigns grades A, B, C, or Fail based on marks input.
6. **Menu-driven Calculator:** Performs addition, subtraction, multiplication, and division based on user choice.
